# Discards

Use this directory for explicit records of failed, merged, abandoned, or pruned directions.

Start from `research/discards/TEMPLATE.md` when logging a discard or failed direction.

Use date-first, lowercase, kebab-case filenames such as `2026-03-27-early-climate-discards.md`.

Good candidates for this folder:

- a weak branch that failed to hold up under grounding
- a set of early directions that were merged into a stronger family
- a comparison pass where one option was clearly abandoned
- a methodological dead end that should not be repeated casually

Discard records should make pruning as legible as successful runs. The point is not to dramatize failure. The point is to keep the search process honest.
