## Discarded Artifacts

### 1. Northern Wheat Correlation Shock

**File:** `research/scenarios/2026-03-27-northern-wheat-correlation-shock.md`  
**Decision:** Merge into parent (synchronous breadbasket stress)  
**Date:** 2026-04-03

**Why discarded:**
- Broad framing overlaps significantly with parent `synchronous-breadbasket-stress.md`
- Less operationally specific than Russia-Europe or Russia-China variants
- Does not add distinct mechanism beyond what parent already covers
- Weakest variant in branch evaluation matrix

**What was learned:**
- Broad scenario families need narrowing to bounded cases to become method-shaping
- Northern wheat as a category is too diffuse; specific node pairs (Russia-Europe, Russia-China) are more useful
- The value of this variant was establishing wheat as the first crop target, which is now superseded by more grounded variants

**Impact on kept work:**
- Russia-Europe wheat trade shock became strongest variant because it has specific trade mechanism
- Southeast Asia rice correlation shock is more generative because it tests crop generalization
- Northern wheat's role is now absorbed into parent scenario as background context

### 2. Synchronous Breadbasket Stress (Parent Scenario)

**File:** `research/scenarios/2026-03-27-synchronous-breadbasket-stress.md`  
**Decision:** Keep as parent, but downgrade from active variant status  
**Date:** 2026-04-03

**Why downgraded:**
- Parent scenario is too broad to be method-shaping
- Structural signal lives in bounded variants, not parent scenarios
- Parent's role is now organizational (groups variants) rather than evidentiary

**What was learned:**
- Parent scenarios are useful for framing but not for method-shaping
- The branch became more useful when it shifted from parent-focused to variant-focused
- This pattern holds across all branches (whiplash, hydrologic, wealth-concentration)

**Impact on kept work:**
- Variants (Russia-Europe, Russia-China, Egypt, Yemen, Philippines, Southeast Asia rice) carry the structural signal
- Parent remains as organizational anchor but not as active comparison target
