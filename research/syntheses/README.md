# Syntheses

Use this directory for higher-order comparisons across notes, scenarios, and experiments.

Use date-first, lowercase, kebab-case filenames such as `2026-03-27-first-scenario-family-synthesis.md`.

Good candidates for this folder:

- scenario-family comparisons
- method reflections across multiple experiments
- emerging conclusions that combine several notes and scenarios

Syntheses should make comparisons explicit and preserve what remains uncertain.
