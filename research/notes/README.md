# Research Notes

Use this directory for working notes, source synthesis, open questions, and partial interpretations.

Use date-first, lowercase, kebab-case filenames such as `2026-03-27-climate-baselines-note.md`.

Good candidates for this folder:

- source reading notes
- emerging hypotheses
- terminology clarifications
- unresolved questions worth revisiting

Notes can be rough, but they should still record sources, assumptions, and uncertainty where possible.
