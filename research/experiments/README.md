# Experiments

Use this directory for tests of the method itself.

Start from `research/experiments/TEMPLATE.md` when creating a new experiment.

Use date-first, lowercase, kebab-case filenames such as `2026-03-27-climate-scenario-generation-pass.md`.

Examples include:

- prompt or workflow experiments
- alternate scenario formats
- evaluation framework tests
- curation process comparisons

Each experiment should state what was tested, how it was evaluated, what changed, and what remains unresolved.
